\section{Starting Criteria}

This contingency procedure is activated if any of these criteria are met for any valve on the TB:
\begin{itemize}
  \item The valve does not go out of the normal state, although it is activated. 
  \item The valve does not go to the normal state, although it is deactivated. 
  \item The valve opens significantly slower or faster than usual.
  \item The valve makes noises that do not comply with the standards. 
  \item The valve does not open or close completely. 
  \item The valve opens or closes without command.
\end{itemize}
